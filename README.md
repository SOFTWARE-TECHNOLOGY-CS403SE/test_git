# test_git
```js
- Lệnh clone source code về
$ git clone <đường dẫn source>
- Lệnh add những thay đổi vừa thực hiện trên source code
$ git add .
- Lệnh ghi chú thích để đẩy lên 
$ git commit -m "test git"
- Lệnh đẩy lên
$ git push
$ git push origin
- Lệnh đẩy lên nhánh
$ git push origin <tên nhánh>
- Lệnh tạo nhánh 
$ git checkout -b nam-branch
- Lệnh chuyển nhánh
$ git checkout <tên nhánh>
```
